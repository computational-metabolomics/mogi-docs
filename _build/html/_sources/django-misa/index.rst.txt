mzml2isa-qt
===========

|Version| |Py versions| |Git| |License| |RTD doc|


mzml2isa-qt is a Graphical User Interface for mzml2isa, that allows
converting mzML files to ISA-Tab files with the help of mzml2isa as well
as adding more metadata to the generated ISA files through its interface.
