django-gfiles
========================================

|Build Status (Travis)| |Py versions|

Simple file management of generic files in Django. Allows files to be saved as symlinks, tracks the original filename,
tracks the user that uploaded the file.

The uploaded files can be viewed and searched using django-tables2 and django-filter applications.
